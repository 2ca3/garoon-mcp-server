# プロジェクト基本情報

- このプロジェクトは Python で書かれた MCPサーバです

# コードスタイル

- 必ず、ツール毎にユニットテストを作して下さい

## コーディング規約

### Python (Ruff適用)

- **行長**: 120文字
- **引用符**: ダブルクォート (`"`)
- **docstring形式**: NumPy スタイル
- **改行コード**: LF (Unix)
- **除外ディレクトリ**: `db_schema/` (Submodule)

**ファイル別ルール:**
- `tests/**`: assert文、マジック値、docstring省略可、日本語関数名許可、パスワード直書き許可
- `scripts/*`: print文許可

# ワークフロー

- 変更完了後は必ず型チェックを実行
- 変更完了後は必ずユニットテストを実行
